Clicking And Dragging
---------------------
- Enable the 'click' and 'drag' functionality on the created shape (rectangle)
    - Pull off 'MouseConstraint' and 'Mouse' objects from the matter.js library

    - Setup a mouse constaint then add it to the 'World' object
        - In order to setup something to respond directly from a mouse input

    - NOTE: The Green color line that appears when clicking & then dragging a shape
        - It represents a spring 
            - Or the force that is applied to the shapeW
