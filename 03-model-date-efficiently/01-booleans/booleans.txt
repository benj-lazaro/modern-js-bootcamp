Boolean
-------
- It is a primitive data type that can ONLY have two values 'true' or 'false'
- They can be stored in a variable
- Reference (https://developer.mozilla.org/en-US/docs/Glossary/Boolean)


Example
-------
- Assigning a boolean value to a variable

let isLoggedIn = true;
let gameOver = false;
const isWaterWet = true;
