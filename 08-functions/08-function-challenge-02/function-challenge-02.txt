Function Challenge 01 Average
-----------------------------
- Write a function to find the average value in an array of numbers
    - It accepts an array of numbers as an argument value
    - It accepts any length of numbers
    - It is assumed to be an array of number; NO value checking required
    - It computes and returns the average value of the numbers
        - Sum each element
        - Divide sum by the number of array elements


Example
-------
avg([0, 50]);               // Returns 25
avg([75, 76, 80, 95, 100]); // Returns 85.2
