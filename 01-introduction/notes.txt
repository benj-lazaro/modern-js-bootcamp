ECMA International
- Organization that creat many different standards for technologies
- Website (https://ecma-International.org)

ECMAScript
- The specification defined in ECMA-262 document for a scripting language
- It is a text document; a set of rules that describes how a language should work
- It is then up to the browsers to go and follow those rules & implement them as JavaScript

TC39 (Technical Committee 39)
- They're in charge of maintaining and evolving the ECMAScript specification
- 40 to 60 delegrates meet 6 times a year to discuss proposed changes to the specification

Can I use
- Online tool that helps determine JavaScript keywords that are supported by modern-day web browsers
- Website (https://caniuse.com/?search=Javascript)

JavaScript
- A programming language that conforms to the ECMAScript specification
